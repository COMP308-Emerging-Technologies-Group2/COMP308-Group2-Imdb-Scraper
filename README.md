# COMP308-Group2-Imdb-Scraper

## Server Address

Server URL: `comp308-group2-imdb-scraper.herokuapp.com`

## Functionality

/GET `/mostPopularMovies` - Returns json object with 'mostPopularMovies' array

/GET `/mostPopularTvs` - Returns json object with 'mostPopularTvs' array
